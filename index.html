Voice & Tone: The Pragmatic Professional

The robonxt voice is built on a single principle: Clarity first, always. Our personality never sacrifices the user's ability to understand. The voice is professional, helpful, and direct, but uses a casual, human tone.

Default Voice (95% of the time)

For all UI copy, instructions, and critical messages. Be clear, concise, and helpful.

"To change your theme, open the settings panel and select 'Appearance'."

The Clever Aside (5% of the time)

For low-stakes moments like success messages or marketing copy. Be witty and context-aware.

"Project saved! All your hard work is safe and sound."

Layout Primitives

The architectural foundation for all pages. These components define structure and hierarchy.

1. Section

A semantic wrapper with no visual style. Its only job is to create vertical spacing between groups of content using margins.

2. Container

The primary, static canvas for main content. It has no animation or hover effects. Use it for the main workspace of a page.

This is a Container. It holds the main content, like this text and the form controls below.

3. Interactive Card

An interactive element used for summaries or links. It has a hover effect to signal that it is clickable.

This is an Interactive Card. Notice the 'lift' effect when you hover over it.
